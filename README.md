# Imposter Game Generator

[![Website](https://img.shields.io/website?url=https%3A%2F%2Fplayimposter.com&label=playimposter.com)](https://playimposter.com/)

**[Play Online Now (No Download Required)](https://playimposter.com/)**

A free, open-source web application for hosting "Who is the Spy" (Imposter) and SpyFall style social deduction games.

## üéÆ Features

*   **Pass & Play Mode**: Use a single device to assign roles. Perfect for offline parties and gatherings.
*   **Customizable Rules**:
    *   Adjust the number of Imposters, Civilians.
    *   Choose from various word categories (Food, Animals, Daily Life, etc.) or use random words.
*   **No App Needed**: Works entirely in the browser on mobile and desktop.
*   **Privacy First**: No account registration required. Game data is stored locally.

## üöÄ Quick Start

1.  Go to the [Live Demo](https://playimposter.com/).
2.  Select **"Generate Your Game"** to start the game.
3.  Follow the on-screen instructions to set up your players and roles.

## üì¶ Installation (For Developers)

To run a local copy of the game generator:

1.  Clone the repository:
    ```bash
    git clone https://github.com/arsterliu/imposter-game-generator.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
